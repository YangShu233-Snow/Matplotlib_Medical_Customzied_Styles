# Context for LLMs

This repository is a collection of matplotlib styles that mimic GraphPad Prism.

## Project Structure
- `/styles/`: Contains all style definitions. Each subdirectory represents a specific chart type.
- `/styles/<chart_type>/assets/`: Contains the `.mplstyle` configuration files.
- `/styles/<chart_type>/example.py`: A complete Python script demonstrating how to use the style with mock data.
- `/styles/<chart_type>/img/`: The output image generated by `example.py`.

## Coding Rules & Instructions for AI Agents
1. **Style Loading:** Always load styles using `plt.style.use(style_file)` before creating the figure.
2. **Path Resolution:** Always use `pathlib.Path` for file paths (e.g., `Path(__file__).parent`) to ensure cross-platform compatibility. Do not use hardcoded string paths.
3. **GraphPad Logic:** GraphPad styles typically require inward-facing ticks, removed top and right spines, and bold axis labels. Rely on the `.mplstyle` files for this rather than writing redundant matplotlib code.
4. **Error Bars:** For biological data, asymmetric error bars (upper only) are preferred.
